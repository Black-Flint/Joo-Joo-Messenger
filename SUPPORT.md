# Support

Thank you for using Joo-Joo Messenger.

## Questions

If you have a question about the project, please use **GitHub Discussions**.

## Bug Reports

If you found a bug, please open a GitHub Issue using the Bug Report template.

## Feature Requests

If you have an idea for improving the project, please open a Feature Request.

## Security Issues

Please do **not** report security vulnerabilities publicly.

See **SECURITY.md** for responsible disclosure instructions.

Thank you for helping improve Joo-Joo Messenger.
