# Roadmap

This roadmap outlines the long-term vision for Joo-Joo Messenger.

The order of features may change as the project evolves. Security, maintainability, and architectural quality always take priority over feature velocity.

---

# Guiding Principles

Every milestone follows these principles:

- Security First
- Privacy by Default
- Maintainable Architecture
- Self-Hosting First
- API First
- Community Driven

No feature should compromise these principles.

---

# Current Status

🚧 The project is currently in active development.

The current focus is building a solid architectural foundation before implementing advanced messaging features.

The following ideas are being explored but are not currently planned.

- Federation
- WebAuthn / Passkeys

---

# Out of Scope

Joo-Joo Messenger will never intentionally include:

- Advertisements
- User tracking
- Vendor lock-in
- Proprietary server requirements
- Closed-source core components
